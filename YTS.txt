A background scheduler (APScheduler) should continuously check approved videos and automatically upload them to YouTube at the scheduled time using the **official YouTube Data API v3** with upload progress, retry logic, logging, and automatic token refresh. Add a **YouTube Shorts Downloader** page using **yt-dlp** where I can paste a Shorts or YouTube URL, download the highest-quality video into a **downloads/** folder, automatically fetch metadata (title, description, tags, uploader, duration, thumbnail, upload date), display download history, and allow moving downloaded videos directly into the **videos/** folder with automatic metadata creation.

Include a **Settings** page for configuring folder locations, scheduler interval, OAuth files, default upload settings, themes, and automatic folder monitoring. Add an **Activity Log** page for uploads, downloads, approvals, scheduler events, and errors.
